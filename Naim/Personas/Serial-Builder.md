## Identity
I've launched 3-5 products. I know the naming process is painful because I've done it multiple times. I'm pragmatic — I know the name matters less than the product, but I also know a bad name creates friction. I'm technical, I move fast, and I don't want to spend more than a couple of hours on this. I've accepted that .com is usually impossible and I'm fine with .io or .co.

## Goal
Get a usable name for my new product in under an hour. Check that it's available across the basics — domain, key social handles. Move on and start building.

## Pain Points (by severity)
1. **I've done this before and it's always the same tedious process.** I know exactly what I need to check and I still have to stitch together 3-4 different tools every time.
2. **Existing checkers are inaccurate.** I've been burned by Namechk showing something as available when it wasn't. I need results I can trust.
3. **I don't need AI hand-holding — I usually have a name idea already.** I just need to verify it fast. Don't make me go through a generation wizard when I already know what I want to check.
4. **Speed matters more than comprehensiveness.** I'd rather check the top 5 platforms quickly than check 100 platforms slowly.
5. **I do this for every new project.** If there was a tool I trusted, I'd use it every time. But nothing's earned that trust yet.

## Workflow (How I Do It Today)
1. Come up with a name based on what the product does — usually takes 30 minutes
2. Check .com, .io, .co on a domain registrar — 2 minutes
3. If .com is taken (usually), check .io or .co — fine with either
4. Quick check on GitHub (my audience is developers), X, and maybe Instagram
5. Pick it and move on. Might rename later if the product gets traction.
6. Total time: 1-2 hours if it goes well, half a day if I get stuck

## What Good Looks Like
I type my name idea. In 10 seconds I see: domain availability across 5 TLDs, social handles on the platforms I care about, and whether there's a company or trademark conflict. Green/red indicators. No fluff. If it's all clear, I'm done. If not, give me 5 alternatives that ARE clear — don't make me brainstorm again.

## Red Flags (I'd Abandon Immediately If...)
- It's slow — if results take more than 15 seconds I'm closing the tab
- It forces me through an AI generation flow when I already have a name
- Inaccurate results — one wrong result and I'll never trust it again
- Requires signup for basic checking — I'll use Namechk instead
- Tries to upsell me before I've seen value

## Voice
Direct, slightly impatient. I value efficiency over polish. I say things like "just tell me if it's available", "I don't need a branding consultation, I need a yes or no", "this is the third time I've done this, I know the drill." I'll pay for a tool that saves me time but I need to see the value first. I respect tools that respect my time.
