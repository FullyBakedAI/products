# Layla.ai Marketing Site — UI Kit

Recreation of the `layla.ai` marketing homepage pattern. All components in this folder are cosmetic reconstructions based on the public site copy and the Layla wordmark; no production code was available.

## Components
- `Nav.jsx` — translucent sticky pill navigation with wordmark + CTA
- `Hero.jsx` — display-face headline, 4 category chips, feature thumbnail
- `DestinationGrid.jsx` — 3-up photo cards for curated trips
- `FeaturePillars.jsx` — 4-up value-prop grid (Tailor-made / Cheaper / Hidden Gems / No Surprises)
- `Testimonials.jsx` — quote cards with initials avatar
- `FAQ.jsx` — collapsible Q&A
- `Footer.jsx` — link columns with "Made with 🩵 in Berlin"

## Running
Open `index.html` directly. Uses React + Babel via CDN. Icons via Lucide CDN.

## Caveats
- No codebase or Figma was provided. Spacing, exact shadow depths, and the display typeface are **informed guesses** based on the live site's copy and wordmark.
- Display typeface (DM Serif Display) is a Google Fonts substitute for the real custom face.
- Photography is placeholder SVG gradients, not real destination photos.
